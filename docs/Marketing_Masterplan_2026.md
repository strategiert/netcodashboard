# TEIL 8: PPC TESTSTRUKTUR

## 🎯 Creative Test-Matrix (pro Marke)

```
            HOOK                    x    PERSONA              x    PROOF
┌─────────────────────────┐        ┌───────────────────┐       ┌──────────────────┐
│ • Risiko/Schmerz        │        │ • Entscheider     │       │ • Zahl/Statistik │
│ • Kosten/ROI            │   x    │ • Anwender        │   x   │ • Case Study     │
│ • Geschwindigkeit       │        │ • IT/Compliance   │       │ • Testimonial    │
│ • Qualität/Sicherheit   │        │ • Einkauf         │       │ • Zertifikat     │
└─────────────────────────┘        └───────────────────┘       └──────────────────┘

= 4 x 4 x 4 = 64 mögliche Kombinationen
→ Pro Monat 4-6 testen, Winner skalieren
```
