# ANHANG: DEFINITIONEN

## Standard-Definitionen (für Prüfbarkeit)

- **dedupliziert** = gleiche Firma/Kontakt zählt im Zeitraum nur einmal
- **qualifiziert** = Use Case passt + Kontaktdaten + echter Bedarf (nicht nur "Info")
- **Unknown-Anteil** = Direct + Unassigned + not set, nur Human Sessions (bot-bereinigt)
- **Standard-Workflow** = wiederkehrender Prozess mit klaren Inputs/Outputs (z.B. Briefing, Report, Newsletter-Versand, Ads-Setup)
- **5-Phasen-Funnel live** = Inhalte + CTAs für alle 5 Phasen vorhanden, intern verlinkt, mit Lead-Magnet-Einstieg
